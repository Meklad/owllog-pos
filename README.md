# Owllog POS

Owllog POS is an open-source mobile app built using ionic, vue.js, capacitor

## Installation

- First clone the repo: 
```
git clone git@github.com:Meklad/owllog-pos.git
```

- Enter the project dirctory, then run:

```bash
npm install
```
